# 스펙 v1 vs v2 성능 비교 실험

두 명세 [`semiconductor_open_price_prediction_spec.md`](./semiconductor_open_price_prediction_spec.md)(v1)와
[`semiconductor_open_price_prediction_spec_v2.md`](./semiconductor_open_price_prediction_spec_v2.md)(v2)를
실제로 구성해 성능을 비교한 결과다. 재현 코드: `experiments/spec_compare.py`
(`python -m experiments.spec_compare`).

> **한 줄 결론.** v1과 v2의 차이는 *알고리즘*이 아니라 **측정 정직성**이다. 실험은
> v2 스펙의 자기 주장 — "더 똑똑한 모델이 아니라 부풀려지지 않은 정직한 측정" — 을
> 정량으로 재현했다. v1의 느슨한 CV는 깨끗한 홀드아웃에서 절반 가까이 무너지고,
> v2의 정직한 측정으로 보면 신호 대부분이 한 줄짜리 `β·SOX` 베이스라인으로 이미
> 설명된다.

---

## 1. 실험 설계 (왜 이렇게 비교했나)

v1·v2는 같은 타깃(`y = log(open[D] / close[D-1])`)과 같은 알고리즘(그래디언트
부스팅)을 쓴다. 그래서 **모델·피처를 고정**하고 v1(느슨)과 v2(엄격)의 **규칙만
토글**해, v1의 낙관적 CV 점수가 깨끗한 홀드아웃에서 얼마나 살아남는지를 측정했다.

| 축 | v1 (느슨) | v2 (엄격) | 근거 |
|----|-----------|-----------|------|
| **정렬** | 같은 캘린더 날짜 join (`allow_exact_matches=True`) → 美 세션 D를 한국 개장 D에 붙임(룩어헤드) | available_at 기반 strict-backward (`=False`) → 직전 밤 美 세션(D-1)만 | v2 §1-2, §4 |
| **분할** | `TimeSeriesSplit(gap=0)`, 홀드아웃 없음, 전체 구간 CV 보고 | `gap=5`(일 단위 embargo) + 최근 12개월 홀드아웃 격리 | v2 §6, §1-9 |
| **타깃** | raw | 학습 fold에서만 winsorize(1%/99%) | v2 §5 |
| **평가** | 공통: `y=0`·`β·SOX` 베이스라인 비교, 방향/조건부 적중률 | 동일 | v2 §7.1, §8 |

### 설치 0개 범위 (yfinance 전용)
- 한국 OHLCV(시초가 포함)는 `005930.KS`/`000660.KS`로 yfinance가 직접 제공
- 美 ★피처: `^SOX, MU, NVDA, AVGO, ^IXIC, ^GSPC, ^VIX, NQ=F, ES=F, DX-Y.NYB, KRW=X`
  (+ 삼성 특화 `TSM, QCOM`)
- 모델: scikit-learn `HistGradientBoostingRegressor`(LightGBM 동일 계열, NaN 네이티브)
- 표본: 2017-01-01 ~ 현재, 종목당 약 2,111 거래일 / 홀드아웃 252일

## 2. 결과

### 2.1 v1의 CV 점수는 홀드아웃에서 무너진다 (낙관 편향)

| 종목 | v1 CV 보고 RMSE | v1 홀드아웃 실제 RMSE | 악화 |
|------|---:|---:|---:|
| 삼성전자 | 1.46% | 2.57% | **+76.7%** |
| SK하이닉스 | 2.04% | 3.17% | **+55.4%** |

embargo·홀드아웃 없이 전체 구간 CV로 보고하면 RMSE가 절반 가까이 좋아 *보인다*.
같은 모델을 격리된 최근 12개월에 적용하면 그 우위가 대부분 증발한다. v2 §6·§1-9가
막으려는 누수가 바로 이것이다.

### 2.2 v2 정직 측정은 베이스라인을 이기지만 폭은 작다

| 종목 | v2 홀드아웃 RMSE | `y=0` 대비 | **`β·SOX` 베이스라인 RMSE** |
|------|---:|---:|---:|
| 삼성전자 | 2.19% | +16.7% | 2.22% |
| SK하이닉스 | 2.45% | +25.8% | 2.65% |

"갭=0" 베이스라인은 분명히 이긴다(+17~26%). 그러나 **한 줄짜리 `β·SOX`(밤사이 SOX
베타) 베이스라인과 거의 동률**이다(삼성 2.19% vs 2.22%). 즉 신호의 대부분은
*"밤사이 SOX가 움직인 만큼 한국 반도체가 따라 연다"*는 거의 기계적인 오버나잇
베타이고, 나머지 피처가 더하는 엣지는 작다(하이닉스에서만 다소 유효).

## 3. 해석

1. **v1 vs v2의 진짜 차이는 알고리즘이 아니라 측정 정직성이다.** 실험이 이를 정량으로
   보여준다(CV→홀드아웃 +55~77% 악화).
2. **v2가 홀드아웃에서 v1보다 오히려 낮은(좋은) RMSE다.** v1의 같은-날짜 join이
   "미래" 美 세션(이미 끝난 한국 갭과 무관)을 붙여 노이즈가 됐기 때문 — v2의 밤사이
   정렬(merge_asof backward)이 인과적으로 옳은 신호를 잡는다. 누수가 항상 성능을
   *부풀리는* 게 아니라, 잘못 정렬되면 *깎아먹기도* 한다는 점도 함께 드러난다.
3. **방향 적중률 ~68–74%는 알파가 아니다.** `β·SOX` 베이스라인도 같은 수준(73%)을
   낸다. 한국 반도체 시초가가 밤사이 SOX를 따라가는 잘 알려진 오버나잇 베타일 뿐이다.
4. 결국 v2 스펙의 전제(§0 주석)대로 **"랜덤워크 대비 의미 있는 엣지가 있는지에 대한
   정직한 답"**은 *"있긴 하나 대부분 오버나잇 SOX 베타로 설명되며, 추가 피처의 한계
   엣지는 작다"*로 정리된다.

## 4. 한계 (이 실험이 못 한 것)

- **yfinance 일봉**이라 야간선물(`NQ=F`/`ES=F`)·환율의 08:59 KST 스냅샷을 못 잡는다
  (v2 §2.1). 따라서 "정직한" v2도 일봉 수준 근사이며, 실시간 인트라데이를 붙이면
  결과가 달라질 수 있다.
- **한국 수급(★ 외국인 순매수, pykrx)·FRED 금리(◆) 미포함.** v1/v2 스펙 모두 외국인
  순매수를 대형주 최강 신호(★)로 본다 — 추가 시 엣지가 커질 여지.
- **LightGBM 대신 sklearn HGB** 사용(동일 그래디언트 부스팅 계열, 단조제약/분위수
  미적용). 알고리즘 자체 비교가 아니라 *측정 프로토콜* 비교가 목적이므로 무방.
- 거래비용·슬리피지 미반영(손익 백테스트 아님, RMSE/적중률만).

## 5. 후속 실험: v2 *모델 파라미터* 적용 (SPEC_IMPROVEMENTS #3)

위 실험은 측정 프로토콜만 토글하고 모델은 HGB로 고정했다. [개선점 #3](./SPEC_IMPROVEMENTS.md)
지적대로, v2 스펙의 **모델링 변경**(§7.2 huber·단조제약·num_leaves 31→15·min_child 30→50·
reg_lambda 1→5·reg_alpha 0→1)을 실제 LightGBM으로 적용해 측정 효과와 분리 검증했다.
재현: `python -m experiments.model_compare` (측정 프로토콜은 v2로 고정, 모델만 토글.
조기종료 셋은 학습 구간 꼬리 15%로 분리해 측정 셋 미접촉).

### 5.1 결과 (홀드아웃 252일 RMSE, 2026-06-11 실행)

| 구성 | 삼성전자 | β·SOX 대비 | SK하이닉스 | β·SOX 대비 |
|------|---:|---:|---:|---:|
| baseline β·SOX | 2.2358% | — | 2.6577% | — |
| HGB(기존 실험) | 2.2182% | +0.8% | 2.4520% | +7.7% |
| LGBM **v1 파라미터** | 2.2179% | +0.8% | 2.5634% | +3.6% |
| LGBM **v2 파라미터** | 2.3917% | **−7.0%** | 2.8018% | **−5.4%** |
| ablation: v2 −huber(→l2) | 2.3917% | (동일) | 2.8018% | (동일) |
| ablation: v2 −단조제약 | 2.3932% | (중립) | 2.8020% | (중립) |
| ablation: v2 −강정규화(v1수준) | **2.2001%** | **+1.6%** | **2.5459%** | +4.2% |

방향 적중률은 반대 방향: LGBM v1/v2 모두 조건부 적중 85%(삼성)로 β·SOX(82%)·HGB(80%)보다
좋다. RMSE 악화와 방향 개선이 공존한다.

### 5.2 발견

1. **v2 모델 파라미터는 RMSE를 오히려 7~9% 악화시킨다.** v2 스펙의 가치는 측정
   프로토콜에 있었고, 모델링 변경은 (이 피처 셋·표본에서는) 역효과다.
2. **huber는 no-op이다.** LightGBM `objective="huber"`의 기본 `alpha=0.9`(δ)는 갭
   타깃 스케일(~1e-2)보다 90배 커서 전 표본이 2차(=L2) 영역에 들어간다 — 예측이 L2와
   소수점 4자리까지 동일했다. 팻테일 대응을 의도한다면 `alpha`를 잔차 스케일
   (예: 0.01)로 내려야 하며, 스펙 §7.2 코드 그대로는 아무 일도 하지 않는다.
3. **단조제약은 중립**(±0.1% 이내). 해롭지 않지만 표본 효율 이득도 측정되지 않았다.
4. **악화의 범인은 강화 정규화다.** num_leaves 15·min_child 50·reg_lambda 5·reg_alpha 1
   조합은 lr=0.02에서 수렴이 너무 느려져 조기종료가 한 번도 발동하지 않고(베스트 반복
   ≈2994/3000, 상한에 봉착) 과소적합 상태로 끝난다. 정규화를 v1 수준으로 되돌리면
   (huber·단조제약 유지) 양 종목 모두 최고 성능이 나온다.
5. **스펙 §7.2 코드의 `subsample=0.8`은 LightGBM에서 `subsample_freq` 미지정 시
   비활성이다** — 실험에서는 두 구성 모두 `subsample_freq=1`을 부여해 동등 비교했다.
   스펙 코드 예시에 반영 필요.
6. 결론적으로 기존 헤드라인("엣지는 대부분 오버나잇 SOX 베타")은 v2 모델 구성으로도
   뒤집히지 않는다. 최선 구성(삼성 +1.6%, 하이닉스 +4.2~7.7%)도 β·SOX 대비 한계
   엣지가 작다는 결론과 일관된다. 단, 이 역시 단일 홀드아웃 점추정이다(개선점 #1·#5).

## 6. 재현

```bash
python -m experiments.spec_compare         # 측정 프로토콜 비교 (v1 vs v2 규칙)
python -m experiments.model_compare        # 모델 파라미터 비교 (v1 vs v2 §7.2) + ablation
python -m experiments.improvements_compare # SPEC_IMPROVEMENTS 적용 후 재검증 (결과: 그 문서 말미)
python -m experiments.full_features_compare # ★-only vs 풀 피처(◆ 확장) 비교 (스펙 §10 8단계)
```

- 코드: `experiments/spec_compare.py`, `experiments/model_compare.py`
- 의존성: `yfinance`, `scikit-learn`, `pandas`, `numpy` (+ model_compare는 `lightgbm`)
- 난수 고정: `RANDOM_STATE=42`. 단, yfinance 최신 시세가 매일 갱신되므로 홀드아웃
  구간 숫자는 실행 시점에 따라 소폭 달라질 수 있다.
